𝟏. ¿𝐐𝐮é 𝐞𝐬 𝐞𝐬𝐭𝐞 𝐩𝐫𝐨𝐠𝐫𝐚𝐦𝐚?

Es un programa hecho en Java que sirve para interactuar con los clientes, flores y diferentes estructuras (colas, pilas, listas, etc.).

Piensa que es como una tienda de flores mágica, pero en tu computadora

𝟐. 𝐂𝐨𝐬𝐚𝐬 𝐪𝐮𝐞 𝐧𝐞𝐜𝐞𝐬𝐢𝐭𝐚𝐬 𝐚𝐧𝐭𝐞𝐬 𝐝𝐞 𝐞𝐦𝐩𝐞𝐳𝐚𝐫
  • Java 8 o más nuevo. Para revisar: 
    java -version
  • Archivos .java del proyecto:

𝟑. 𝐏𝐫𝐞𝐩𝐚𝐫𝐚𝐫 𝐥𝐚 𝐜𝐚𝐫𝐩𝐞𝐭𝐚

1. Crea una carpeta en tu Escritorio llamada Floresapp.

2. Mete todos los archivos .java dentro de esa carpeta.

𝟒. 𝐂𝐨𝐦𝐩𝐢𝐥𝐚𝐫 𝐞𝐥 𝐩𝐫𝐨𝐠𝐫𝐚𝐦𝐚

Abre la terminal y escribe:

cd ~/Desktop/FloresApp
javac *.java

Si no aparece nada, significa que todo salió bien

𝟓. 𝐄𝐣𝐞𝐜𝐮𝐭𝐚𝐫 𝐞𝐥 𝐩𝐫𝐨𝐠𝐫𝐚𝐦𝐚 
java Main

Verás el menú principal del programa.

𝟔. ¿𝐂ó𝐦𝐨 𝐢𝐧𝐭𝐞𝐫𝐚𝐜𝐭ú𝐚𝐫 𝐜𝐨𝐧 𝐞𝐥 𝐜𝐨𝐝𝐢𝐠𝐨?
  • 1 → Si eliges la opción 1, podras realizar pedidos. Se pedirá el nombre del cliente, número, dirección, tipo de flores y si el pedido es urgente.
  • 2 → En el número 2, puedes eliminar cualquier pedido. Solo introduce el nombre del cliente, si no hay clientes registrados, aparecerá una advertencia.
  • 3 → Si quieres ver toda la lista y catalogos de flores, la opción 3 te muestra el catálogo de flores junto con sus nombres y precios.
  • 4 → En la opción 4, se muestran todos los pedidos completados.
  • 5 → Aquí podrás ver pedidos en tres estados: pendientes, realizados y futuros.
  • 6 → Observa los pedidos que están en proceso (pendientes de completar).
  • 7 → Aqui basicamente se le muestra al usuario un arbol binario con toda la información recopilada de los clientes.
  • 8 → En este apartado solo le das click e ingresas el nombre del cliente que buscas junto a su información.
  • 0 → Cierra el sistema.
𝟕. 𝐄𝐣𝐞𝐦𝐩𝐥𝐨 𝐝𝐞 𝐥𝐚 𝐢𝐧𝐭𝐞𝐫𝐟𝐚𝐳 
                    ===== MENÚ PRINCIPAL =====
        | 1. Realizar pedido                            |
        | 2. Eliminar pedido por nombre                 |
        | 3. Consultar catálogo de flores               |
        | 4. Mostrar todos los pedidos                  |
        | 5. Ver historial de pedidos                   |
        | 6. Procesar pedido                            |
        | 0. Salir

Selecciona el número y presiona Enter en la terminal.

𝟖. 𝐂𝐨𝐧𝐬𝐞𝐣𝐨𝐬
  • Si te equivocas con un número, vuelve a elegir la opción correcta.
  • Siempre que quieras parar, escribe 0.
  • No necesitas internet para usarlo — solo Java y tu computadora
