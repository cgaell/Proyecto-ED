Manual de usuario
𝟏. ¿𝐐𝐮é 𝐞𝐬 𝐞𝐬𝐭𝐞 𝐩𝐫𝐨𝐠𝐫𝐚𝐦𝐚?

Es un programa hecho en Java que sirve para interactuar con los clientes, flores y diferentes estructuras (colas, pilas, listas, etc.).

Piensa que es como una tienda de flores mágica, pero en tu computadora

𝟐. 𝐂𝐨𝐬𝐚𝐬 𝐪𝐮𝐞 𝐧𝐞𝐜𝐞𝐬𝐢𝐭𝐚𝐬 𝐚𝐧𝐭𝐞𝐬 𝐝𝐞 𝐞𝐦𝐩𝐞𝐳𝐚𝐫
  • Java 8 o más nuevo. Para revisar: 
    java -version
  • Archivos .java del proyecto: 
    Stack.java
Queue.java
PriorityQueue.java
PriorityNode.java
Cliente.java
Flores.java
HashTable.java
Node.java
Floresapp.html
SimpleLinkedList.java
Main.java
𝟑. 𝐏𝐫𝐞𝐩𝐚𝐫𝐚𝐫 𝐥𝐚 𝐜𝐚𝐫𝐩𝐞𝐭𝐚

1. Crea una carpeta en tu Escritorio llamada Floresapp.

2. Mete todos los archivos .java dentro de esa carpeta.

𝟒. 𝐂𝐨𝐦𝐩𝐢𝐥𝐚𝐫 𝐞𝐥 𝐩𝐫𝐨𝐠𝐫𝐚𝐦𝐚

Abre la terminal y escribe:

cd ~/Desktop/FloresApp
javac *.java

Si no aparece nada, significa que todo salió bien

𝟓. 𝐄𝐣𝐞𝐜𝐮𝐭𝐚𝐫 𝐞𝐥 𝐩𝐫𝐨𝐠𝐫𝐚𝐦𝐚 
java Main

Verás el menú principal del programa.

𝟔. ¿𝐂ó𝐦𝐨 𝐢𝐧𝐭𝐞𝐫𝐚𝐜𝐭ú𝐚𝐫 𝐜𝐨𝐧 𝐞𝐥 𝐜𝐨𝐝𝐢𝐠𝐨?
  • 1) Realizar pedido → Escribe nombre, teléfono, dirección, flor y si es urgente. El sistema crea un ID automático y guarda fecha y hora.
  • 2) Eliminar pedido por ID → Escribe el ID del pedido para borrarlo si existe.
  • 3) Consultar catálogo de flores (con filtros) → Elige: 1 (Mayor a menor), 2 (Menor a mayor), 3 (Default).
  • 4) Mostrar todos los pedidos → Lista todos (pendientes).
  • 5) Ver información de la empresa → Muestra un árbol con departamentos/roles.
  • 6) Buscar pedido por ID → Escribe el ID y te muestra los datos del pedido.
  • 7) Procesar pedido por ID → Marca el pedido como realizado y guarda la fecha/hora.
  • 8) Ver historial de pedidos procesados → Lista los pedidos ya realizados con su fecha.
  • 9) Ver ganancias totales → Muestra la suma $ de los pedidos procesados.
  • 0) Volver → Regresas al menú anterior cuando quieras.

𝟖. 𝐂𝐨𝐧𝐬𝐞𝐣𝐨𝐬

  • Siempre que quieras parar, escribe 0.
  • No necesitas internet para usarlo — solo Java y tu computadora
